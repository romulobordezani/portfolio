Squaretrade Go

Appointments listing Page, where Technicians can find out new scheduled appointments and all information needed, like parts to replace on the cellphones, manuals, tools and so on.

Main challenges:
  • UX / UI improvements made by me, using Sketch and Invision.
  • Dissociate duplicated styles from the core MVC made with Flask and Python to a Sassy way, using Angular.
  • Filter appointments based on status + dates, like finished, on going, about to happen, tomorrow, etc..
