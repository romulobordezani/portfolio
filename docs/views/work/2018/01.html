GFG's Shared Components Library made with React

A NPM library to share UI components, supporting themes in a centralized and distributed way, but safe while publishing new updates.

Main challenges:

  • To deploy new versions over a private NPM (Verdaccio) without harm old client projects
  • Generate a solid and integrated way to document the components and it's props, in a way where collaborators could use README.md files
  • Support Multiple Themes for diferente Stores.
  • Support Multi-languages and 1i18n
  • Use only .Jss due to server renders performance issues with CSS styles
  • Do not overload with the whole Material Ui library, but support close features and way to set up stuff
  • To use the new React's Context API to spread themes, once upon not all projects are currently using Redux.
