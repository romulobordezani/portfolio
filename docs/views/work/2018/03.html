Dafiti's new product page PWA

React mobile version of the Dafiti's product page, available Demo (Mobile Only)

Main challenges:

  • Activate service workers trough several servers and APIs over the same domain (In progress)
  • Crop, lazyload and cache images via Akamai using Thumbor for high audiences.
  • React Carousel with full support to pinch and touches using synthetics events.
  • Server Side React Rendering with Electrode.io to achieve a better SEO.
  • Back-end for front-end using Hapi.js as middleware to connect to different APIs, for different countries and stores.
