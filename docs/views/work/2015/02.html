Development part of this site.

All built with Google's Angular and Material Angular, it's a jQuery less site to save memory for mobile devices.

Uses Animate.css to power CSS3 animations and transitions, all made in HTML5 standards.

Main challenges:

  • To avoid jQuery and use only Angular and native JavaScript to gain performance over mobile devices
  • To provide Unit tests environment with Karma and Jasmine
  • To integrate Jenkins to Continuous Deployment
  • To use jsHint as code style
  • To mix up touch events and mouse ones in the same app
  • To evolve md-tabs, that doesn't select the tab when the user swipe to the content
  • To consume Google Geo location Information System to get the user Address automatically
  • To build up a dynamic timeline based on years to filter the portfolio
  • To use node.js and Grunt to manage the app distribution and life cicle
  • To implement Bower to control external dependencies
  • To supply a way to update the browser when any file is saved during the development, in case the live connect
  • To pack it in a hybrid web app for IOS, Android and Windows 8.
  • To support retina devices and high density screens
