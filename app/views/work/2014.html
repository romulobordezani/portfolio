Appendit

Appendit

Grunt plugin based on grunt-cdn, that was evolved to append strings as params at cdnfieds urls.

At UOL, they use a CDN that controls the file version within a url param.

The official grunt cdn doesn't do it, so I evolved it to append parameters after CDNfy.

This is available at Node.js npm official repository, and I built it at home in my free time, so you can use at your will.
