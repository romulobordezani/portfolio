User Expirience and layout design of this site

I met Material Design in a Google Developers Group conference at Google Brasil, and the guy who was talking about it was so convincing that inspired me to entirely rebuild my own site using their style guide.

Let's start talking about the logo, it's inspired in the Da Vinci's Forster Codex in a tribute to Leonardo, because he was a (de)coder and also was able to draw and paint wonderfully.

UX chalenges and requirements:

  • To organize my jobs by years, evidencing these 13 years of experience working with web and digital communication.
  • To be full responsive to support cellphones and tablets.
  • To integrate css animations between pages and user actions, respecting each movement unit.
  • To use the collor pallete in all the unit interfaces.
  • To keep everything adherent to the Material Design Style Guide, also called Google's Visual Frame Work.
  • To be able to collect any user feedback, to measure it and start the Lean process with the "Learn" stage, after my first "Hypothesis" launched.
  • To build up a form without aggressive error messages, if was possible, without any error messages, just using colors and restricting submit.

Design steps:

  • I did the Layout over the first paper wireframe, using Fireworks, Illustrator and Photoshop.
  • Vectorized each icon and draw that could be exported in .svg format.
  • Now I keep evolving the design and the responsiveness directly into the code.
