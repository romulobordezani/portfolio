CSS Dinâmica

Objetivos

  • Aprender o que é CSS na prática com uma dinâmica
  • Compreender como funcionam seletores e a síntaxe básica do CSS

O que é CSS?

CSS é o conjunto de regras que dirão à uma página web qual será seu formato, seu tamanho, posição, cores e etc.

Para entendermos melhor o uso dos estilos, simularemos o trabalho de um escultor tentando realizar sua obra prima, conforme o esboço a seguir:

Como instruir nosso modelo a fazer a pose necesssária para nossa obra prima?

Sintaxe básica do CSS

seletor {
      propriedade: valor;
}

IDs vs Classes

Vamos identificar as mãos do nosso modelo como:
mao-esquerda e mao-direita

Memo:

Para se lembrar do caracter usado para selecionar IDs, a #, lembre-se que ela parece uma cadeia e que na cadeia ninguém pode ficar preso sem ser identificado.

O código ficaria assim:

#mao-esquerda {
      posição: ao-alto;
}
#mao-direita {
      posição: ao-alto;
}

Mas e se quisermos dizer ao modelo para levantar ambas as mãos de uma vez?

Como podemos selecionar mais de um elemento ao mesmo tempo para passar instruções?

Para isso podemos usar as classes

As classes são marcações que usam o caracter . como prefixo.

Memo:

Para ajudar a memorizar o prefixo das classes, lembre-se do . em comum entre os elementos que quer selecionar.

O código para as classes ficaria assim:

.mao {
      posição: ao-alto;
}

As .classes são úteis para reutilizar definições de estilo


Os #Ids são úteis para isolar definições de estilo

Exemplo de isolamento: repare novamente a mão esquerda mais alta no esboço

.mao {
      posição: ao-alto;
      cor: cinza;
      musculatura: sarada;
}
#mao-esquerda {
      posição: ainda-mais-ao-alto;
}
